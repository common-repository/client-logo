== Description ==
= Logo slider =

You need a slider for logo/images? Then this plugin is for you!

This plugin is created for logo slider.

= Features =

1. Display unlimited number of WordPresss custom posts slider in a single page or post with different sets of options like category.
2. Edit, Update & Delete any time.
3. WordPress Logo slider displays your custom posts using various designs.
4. Touch for mobile devices and responsive.

== Installation ==
1. Unzip and upload the `logo slider` directory to the plugin directory ('/wp-content/plugins/')
2. Activate the plugin through the 'Plugins' menu in WordPress
